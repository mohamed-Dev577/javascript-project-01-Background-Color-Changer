# 🎨 Background Color Changer

مشروع بسيط باستخدام JavaScript يقوم بتغيير لون خلفية الصفحة عند الضغط على زر.

هذا المشروع يعتبر من أول المشاريع التي يتعلمها المبتدئون لفهم كيفية التفاعل بين JavaScript و HTML.

---

## 🚀 فكرة المشروع

الصفحة تحتوي على:

- عنوان
- زر
- عند الضغط على الزر يتم تغيير لون خلفية الصفحة

---

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript

---

## 🎯 ماذا تعلمت من المشروع؟

- ربط JavaScript بملف HTML
- استخدام `getElementById` للوصول إلى عنصر
- التعامل مع حدث الضغط `onclick`
- تغيير خصائص CSS باستخدام JavaScript
